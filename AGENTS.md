# AGENTS.md

## Goal

Personal Neovim configuration.

Keep the configuration:

- small
- understandable
- maintainable

## Rules

- Prefer built-in Neovim features.
- Avoid unnecessary plugins.
- Create ADRs for major decisions.
- Make small changes.
- Explain trade-offs.
